# 📰 GenAI News Reporting using OpenAI API

This project leverages **Generative AI** to automate news reporting by extracting information from official government sources, generating concise summaries, and creating structured news articles using the **OpenAI API**.

## 📊 Overview

Preparing news articles from lengthy press releases and government publications can be time-consuming. This project streamlines the process by using Large Language Models (LLMs) to summarize content, paraphrase key information, and generate publish-ready news reports, enabling faster and more efficient news creation.

## 🔍 Key Features

* 📰 AI-powered news article generation
* 📄 Automatic summarization of government press releases
* ✍️ Intelligent paraphrasing for improved readability
* 🌐 Content extraction from official government websites
* 🤖 News generation using the OpenAI API
